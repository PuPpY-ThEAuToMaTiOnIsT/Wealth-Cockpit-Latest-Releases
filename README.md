## Installation

Wealth Cockpit is built with Python and Flet.

### Prerequisites
*   Python 3.7 or higher

### Get Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/wealth-cockpit.git
    cd wealth-cockpit
    ```

2.  **Install dependencies:**
    ```bash
    pip install flet
    ```

3.  **Run the application:**
    ```bash
    python main.py
    ```

## Usage Guide

1.  **Income Tab**: Enter your monthly salary, bonuses, and other income sources.
2.  **Expenses**: Input your monthly expenses, EMIs, and insurance premiums. This drives the FIRE calculations.
3.  **Assets Tabs (Equity, Debt, etc.)**: Input the current value of your investments.
4.  **Dashboard**: Watch your Net Worth and FIRE progress update automatically!
